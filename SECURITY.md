# Security Policy (Draft)

## Non-negotiables

- Provider keys must never be stored in Skyrim scripts or mod files.
- The local runtime runs on localhost by default.
- No remote access unless explicitly enabled by the user.
- Logs and replay bundles are **redacted by default**.

## Reporting

This is a draft security policy for an early-stage reference project.
If you find a vulnerability, open a private issue or contact the maintainer.
